# Customer Support Chatbot

Lightweight customer success assistant built with React + Vite. The UI focuses on a calm color palette, responsive layout, and quick actions to speed up common support workflows.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Visit http://localhost:5173 to chat with Clara. You can trigger scripted intents via the suggestion chips or type free-form questions.

## Scripts

- `npm run dev` – start the Vite dev server with fast HMR
- `npm run build` – bundle the app for production
- `npm run preview` – preview the production build locally

## Tech Stack

- React 18 with hooks for state & interaction
- Vite 5 for a fast tooling experience
- Modern CSS for layout, animations, and responsive behavior
